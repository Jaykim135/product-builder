# Powerball Lottery Drawer

## Overview

This application is a simple Powerball lottery number generator. It allows users to simulate a Powerball drawing by clicking a button, which will generate and display five random white balls and one red Powerball. The site also includes informational sections about Powerball and essential policy pages to ensure compliance and improve chances for AdSense approval.

## Implemented Features

*   **UI:**
    *   A clear and simple interface with a title, a container for the lottery balls, and a "Draw" button.
    *   The lottery balls are styled to look like actual Powerball tickets.
    *   The layout is responsive and centers the content.
    *   **Dark Mode Toggle:** A switch to enable/disable dark mode for improved user experience.
    *   **Informational Sections:** Added "About Powerball" and "How to Play" sections with detailed content.
    *   **Footer:** Included a footer with links to "Privacy Policy" and "Terms of Service".
*   **Functionality:**
    *   Clicking the "Draw Numbers" button generates 5 unique random numbers between 1 and 69 for the white balls and 1 random number between 1 and 26 for the red Powerball.
    *   The generated numbers are displayed in the UI.
    *   **Disqus Comments:** Integrated a Disqus comment section for user engagement.
